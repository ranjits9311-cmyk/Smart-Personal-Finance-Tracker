# Smart Personal Finance Tracker

A lightweight personal finance tracker built with **vanilla HTML/CSS/JS**.

## Features

- Add income/expense transactions with **category** + **date**
- Live totals: **Balance / Income / Expenses**
- Search/filter transactions
- Delete individual transactions
- **LocalStorage persistence** (keeps your data after refresh)
- Export all transactions to **CSV**
- Backup / Restore using **JSON** (data safety)

## Live demo

https://ranjits9311-cmyk.github.io/Smart-Personal-Finance-Tracker/

## Run locally

Just open `index.html` in your browser.

## Notes

- Data is stored locally in your browser (LocalStorage). Clearing browser storage will remove transactions.
